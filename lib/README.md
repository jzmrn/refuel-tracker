# Shared Library

This directory contains shared library components for the refuel tracker project.

## Tankerkoenig API Client

A Python client for accessing the Tankerkoenig API to retrieve gas station price data in Germany. See the [README](tankerkoenig/README.md) for detailed documentation.

## Metrics

A Python client to store and retrieve metric data using SQLite and Parquet. See the [README](./metrics/README.md) for detailed documentation.
